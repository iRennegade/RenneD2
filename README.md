# RenneD2
Una tool DDoS / DoS Open Source


# Install
## Linux
```
sudo chmod +x install.sh
```
Despues se debe ejecutar esto
```
./install.sh
```

## Windows

```
start install.bat
```

# Atacar

### DDoS
Esto es premium, más información a el Discord oficial de HackerSquad

### DoS
```
python main.py <url>
```
